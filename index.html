1day-learn 기본 문법 tag

가장 중요한 tag 사용법을 배웁니다.
먼저, 두가지 검색을 통해 학습하는 방법을 배워봅니다.

첫번째, 검색을 하여 tag 사용법을 학습합니다
- html tag를 검색하여 샘플들을 적용하여 직접 공부하는 것을 추천한다.
- ex) html h1 tag 를 검색하여 살펴본다.

두번째, 통계를 분석하여 150+ tag를 효과적으로 학습합니다.
- www.advancewebbanking.com/html/에서 가장 빈도수가 많은 tag를 알 수 있습니다.
- 외국어 공부시에도 통계를 활용하여 가장 많이 사용되는 단어를 먼저 배우면 효과적이다. ex)frequency chinese words 를 검색하여 본다.

1.h1~h6 tag 사용법 및 학습하기
2.p(paragraper), br tag 사용법 학습하기
3.줄 간격을 세밀하게 조정하기위한 css 언어 맛보기
4.p 다음에 p style="margin-top:40;" 삽입하고 숫자 40을 다양하게 변경하는 학습한다.

1day-html이 중요한 이유

검색 엔진은 모든 웹페이지의 html을 분석하여 tag을 근거로 해서 정리 정돈을 합니다.
비지니스 측면에서 검색엔진은 제목이 코딩인 사이트를 먼저 노출하게 되므로 html을 활용해서 웹페이지을 만드는 것이 중요하다.
또한 휴머니즈 측면에서 웹은 모든 운영체제와 장애인에 접근성이 개방되어 있다는 것이다.

1day-속성 : img

이미지를 넣은 작업을 배웁니다
참고 : 저작권이 없는 고품질 사진 사이트 - unsplish

1. 사진을 folder안에 저장을 합니다.
2.사진 파일 이름을 간단하게 재지정 합니다.
3.img tag를 치고 그 안에 src tag를 쳐서 파일이름을 써 줍니다.
4.사진 사이즈를 width tag를 써서 크기를 지정하거나 100%로 지정해 줍니다.

2day-parent, child tag

목록을 만들어 봅니다

1.목록을 뚯하는 list = li tag를 활용하여 리스트를 순서대로 만듭니다.
2.li tag는 ul tag를 parent로 가지고 있다.
3.ul = unordered list를 뜻함
4.자동으로 순서를 정리하게 하기 위해서는 ol tag를 사용하면 앞에 순서대로 리스트가 만들어집니다.
